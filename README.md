# 👕 Meik – Buy & Sell Clothes Marketplace

Meik is a mobile marketplace application that allows users to buy and sell both new and used clothes easily. The platform connects buyers and sellers in a simple and secure environment where users can list products, browse clothing items, and communicate directly.

---

## 📌 Project Information

### Project Name
Meik

### Short Project Description
Meik is a mobile marketplace app that enables users to sell and buy new or second-hand clothes. Users can upload product listings, explore available items, and connect with other users for smooth transactions.

### Industry / Use Case
E-commerce / Marketplace / Fashion

---

## 🚀 Key Features

- 👗 **Sell Your Clothes**  
  Users can upload photos and details to list new or used clothes for sale.

- 🔍 **Browse Products**  
  Explore a wide variety of clothing items from different sellers.

- 🛍 **Buy Items Easily**  
  View product details, prices, and seller information.

- 💬 **Chat with Sellers**  
  Communicate directly with sellers before purchasing.

- ❤️ **Favorite Items**  
  Save products to your wishlist for later.

- 📱 **User-Friendly Interface**  
  Clean and simple design for easy browsing and selling.

---

## 🛠 Technology Stack

- **Flutter** – Cross-platform mobile development  
- **Dart** – Programming language  
- **REST API** – Backend communication  
- **Cloud Storage** – Image uploads for products
